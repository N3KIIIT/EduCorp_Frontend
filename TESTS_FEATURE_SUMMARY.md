# Функционал тестов для курсов

## Обзор
Добавлена возможность создания и прохождения тестов для каждого курса. На карточке курса добавлена вкладка "Тесты", на которой можно создавать и проходить тесты.

## Структура проекта

### Новые компоненты

#### 1. Список тестов (`src/features/education/manage-tests/ui/TestList.tsx`)
- Отображает список тестов для выбранного курса
- Позволяет добавлять вопросы к тесту
- Поддерживает удаление тестов
- Показывает информацию о тесте (тип, количество вопросов, проходной балл)

#### 2. Детали теста (`src/features/education/manage-tests/ui/TestDetailsPanel.tsx`)
- Отображает подробную информацию о тесте
- Содержит две вкладки:
  - **Вопросы**: список вопросов теста
  - **Пройти тест**: инструкции и кнопка для начала прохождения

#### 3. Список вопросов (`src/features/education/manage-tests/ui/QuestionList.tsx`)
- Отображает список вопросов для выбранного теста
- Показывает тип вопроса, текст и баллы
- Поддерживает редактирование и удаление вопросов

#### 4. Прохождение теста (`src/features/education/manage-tests/ui/TestTakePanel.tsx`)
- Интерактивный интерфейс для прохождения теста
- Поддержка различных типов вопросов:
  - Одиночный выбор (Single Choice)
  - Множественный выбор (Multiple Choice)
  - Текстовый ответ (Text)
  - Числовой ответ (Numeric)
- Таймер для ограниченного времени прохождения
- Индикатор прогресса
- Навигация между вопросами (Вперед/Назад)
- Отправка результатов на сервер

## Функциональность

### Для администраторов/менеджеров:
1. **Создание тестов**:
   - Выбор типа теста (практический/контрольный)
   - Установка проходного балла
   - Настройка длительности
   - Настройка политики прохождения (перемешивание вопросов, показ правильных ответов, пауза)

2. **Управление вопросами**:
   - Создание вопросов разных типов
   - Установка баллов за вопрос
   - Добавление описания, подсказок и объяснений
   - Редактирование и удаление вопросов

### Для студентов:
1. **Просмотр тестов**:
   - Список доступных тестов в курсе
   - Информация о тесте (тип, длительность, проходной балл)

2. **Прохождение тестов**:
   - Последовательное прохождение вопросов
   - Таймер с обратным отсчетом
   - Индикатор прогресса
   - Возможность вернуться к предыдущим вопросам
   - Отправка результатов

## Технологии и подходы

### Архитектура
- Использован подход **feature-sliced design** для организации кода
- Разделение на слои: `features`, `entities`, `widgets`, `shared`
- Каждый модуль (тесты, вопросы) имеет свою структуру:
  - `api/` - API-хуки для работы с данными
  - `ui/` - UI-компоненты

### Состояние
- Использование `react-query` для управления состоянием и кэширования данных
- Глобальное состояние навигации через `useNavigationStore`
- Контекст приложения через `useAppContextStore`

### UI/UX
- Использование компонентов VKUI для согласованного интерфейса
- Адаптивный дизайн
- Интуитивная навигация с вкладками
- Визуальные индикаторы прогресса и состояния

## Как использовать

### Создание теста:
1. Перейти в карточку курса
2. Переключиться на вкладку "Тесты"
3. Нажать кнопку "Добавить тест"
4. Заполнить форму создания теста
5. Добавить вопросы через кнопку "Добавить вопрос"

### Прохождение теста:
1. Перейти в карточку курса
2. Переключиться на вкладку "Тесты"
3. Выбрать нужный тест
4. Перейти на вкладку "Пройти тест"
5. Нажать "Начать тест"
6. Отвечать на вопросы, используя навигацию "Назад"/"Далее"
7. Нажать "Завершить тест" для отправки результатов
